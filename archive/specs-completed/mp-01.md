# MP-01: Mağaza Profili Genişletme — 4 Yeni Alan

**Durum:** Açık
**Öncelik:** P2
**Tahmini süre:** 2 saat

---

## Amaç

Marka profiline ek alanlar ekleyerek üretim kalitesini artırmak. Mevcut: marka_adi, ton, hedef_kitle, vurgulanan_ozellikler.

## Yeni alanlar

### 1) Mağaza kategorileri
- `profiles` tablosuna `magaza_kategorileri text[]`
- Multi-select chip: Kadın Giyim, Erkek Giyim, Çocuk, Kozmetik, Elektronik, Ev & Yaşam, Gıda, Takı & Aksesuar, Spor, Oto, Kitap & Hobi, Diğer
- Max 3 kategori
- Prompt: "Bu mağaza şu kategorilerde satış yapıyor: [kategoriler]"

### 2) Fiyat bandı
- `profiles` tablosuna `fiyat_bandi text`
- 3 seçenek: Ekonomik (0-100 TL) / Orta (100-500 TL) / Premium (500 TL+)
- Prompt: "Mağazanın fiyat bandı: [band]"

### 3) Teslimat/hizmet vurguları
- `profiles` tablosuna `teslimat_vurgulari text[]`
- Checkbox: Hızlı kargo, Kolay iade, Yerli üretim, Organik/doğal, Hediye paketi, Garanti, Taksit imkanı
- Prompt: "Bu mağazanın vurguladığı değerler: [seçilenler]"

### 4) Rakip/benchmark mağaza (opsiyonel)
- `profiles` tablosuna `benchmark_magaza text`
- Text input: "Beğendiğin veya hedeflediğin bir mağaza var mı?"
- Prompt: "Kullanıcı şu mağazayı referans alıyor: [isim]"

## Migration

```sql
alter table profiles
  add column if not exists magaza_kategorileri text[],
  add column if not exists fiyat_bandi text,
  add column if not exists teslimat_vurgulari text[],
  add column if not exists benchmark_magaza text;
```
